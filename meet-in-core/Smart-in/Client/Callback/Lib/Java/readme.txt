　Smart-in　API　for Java　の導入方法

■内容物
・smart-in-api.jar　APIファイル


■設定手順

1)　ご利用のWEBサービスのビルドパスに smart-in.jar を設定してください。

2)　sin.key をご利用のWEBサーバ内の任意の場所に設置してください。

3)　設置したsin.key に対し、必要に応じ内容を変更してください。
　　code=******************（Smart-inより提供されたキーデータ）
　　svr=接続先Smart-inサーバ（ドメイン）

4)　コーディングの際、SinConnectorクラスのインスタンス生成時に
　　パラメータに (2)で設置したファイルの絶対パスを指定してください。
